# earth2studio-polar

Sea-ice and Southern Ocean extensions for [NVIDIA Earth2Studio](https://github.com/NVIDIA/earth2studio):
polar data sources, satellite-altimetry-based sea-ice diagnostics, and standing-meander metrics.

This is an independent research project and is not affiliated with or endorsed by NVIDIA.

## Status
Early development. Interfaces track earth2studio >= 0.15 and may change.
